Sobre mim

Desde criança gostei de ciências exatas e trabalhar de forma criativa para solucionar problemas. Mais tarde, percebi que a área da computação tem os atrativos e desafios que gosto.
Sempre busco aprender coisas novas e manter-me atualizado. Valorizo o conhecimento e o esforço. Almejo fazer grandes contribuições e ser lembrado por ter participado delas.

  • Desenvolvimento Web
    Ícone relógio Duração do curso 107 horas Ícone calendário Período de realização Atualmente cursando

    O curso Desenvolvimento Web Completo Link externo é oferecido pela plataforma de ensino online Udemy. Dividido em 20 módulos, o curso contém mais de 550 aulas, abordando temas desde o front-end até o back-end. É ensinado HTML 5, CSS 3, Bootstrap 4, Javascript, ECMAScript, Programação Orientada a Objetos, PHP 7, MySQL, Ajax, JQuery, APIs, SASS, IONIC e Wordpress.   • Applied Data Science with Python
    Duração do curso 23 horas Período de realização Atualmente cursando

    O curso Applied Data Science with Python Link externo é oferecido pela plataforma de ensino online Cognitive Class - IBM. O curso é um conjunto de outros 3 cursos, que são Python for Data Science, Data Analysis with Python e Data Visualization with Python. Todos os 3 cursos possuem avaliações, em formas de questionários, feitas utilizando Jupyter Notebook no JupyterLab.

    Python for Data Science
    Duração do curso 5 horas Período de realização Atualmente cursando

    O curso é dividido em 5 módulos, abordando desde tipos de dados, estruturas de seleções, loops, classes e objetos, manipulação de arquivos, utilização da biblioteca Pandas e do pacote Numpy.

    Data Analysis with Python
    Ícone relógio Duração do curso 8 horas

    O curso é dividido em 5 módulos, que são Importando Datasets, Limpando e preparando os dados, Resumindo o Data Frame, Desenvolvimento de modelo e Avaliação de modelo.

    Data Visualization with Python
    Ícone relógio Duração do curso 10 horas

    O curso é dividido em 5 módulos, que são Introdução às ferramentas de visualização, Ferramentas básicas de visualização, Ferramentas especializadas de visualização, Ferramentas avançadas de visualização e Criando mapas e visualizando dados geoespaciais.
